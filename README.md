# Weather Dashboard

The Weather Dashboard is a web application that displays the weather forecast for multiple days. It provides information such as the day, weather conditions, maximum temperature, and wind speed. Users can also view hourly temperature and wind data for a selected day.

## Features

- View the weather forecast for multiple days
- Display detailed information for each day, including weather conditions, maximum temperature, and wind speed
- Toggle between days to view hourly temperature and wind data
- Heatwave and strong wind warnings displayed when applicable

## Technologies Used

- React: JavaScript library for building user interfaces
- Recharts: Charting library for React
- CSS: Styling the user interface
- JavaScript: Programming language for interactivity and data manipulation

## Installation

1. Clone the repository: `git clone <repository-url>`
2. Navigate to the project directory: `cd weather-dashboard`
3. Install the dependencies: `yarn install`

## Usage

1. Start the application: `yarn start`
2. Open your web browser and visit `http://localhost:3000`
3. The Weather Dashboard will be displayed, showing the weather forecast for multiple days
